# Aegis Trade AI Security Notice

No offline licensing system is uncrackable. This system uses device-bound signed activation keys to prevent casual sharing and unauthorized activation.

The app does not claim perfect anti-piracy. It does not provide global online revocation without a hosted backend, and it cannot destroy an activation key after that key has been sent to a user.

Never share Telegram bot tokens, private signing keys, admin PINs, or unencrypted admin backups.
